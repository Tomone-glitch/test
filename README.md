# 任务看板 (Kanban Board)

一个简洁的 Trello 风格三列看板，纯前端实现，无需任何依赖。

## 功能特点

- **三列看板** — 待办、进行中、已完成，任务状态一目了然
- **拖拽操作** — 支持鼠标拖拽（桌面端）和触摸拖动（移动端），拖拽时半透明效果 + 目标列高亮提示
- **移动按钮** — 每张卡片提供快捷移动按钮，点击即可切换任务状态
- **本地存储** — 使用 localStorage 持久化数据，刷新或关闭页面后任务不丢失
- **响应式设计** — 适配电脑、平板和手机，窄屏自动切换为纵向单列布局
- **零依赖** — 纯 HTML、CSS、原生 JavaScript，无需安装任何工具

## 技术栈

- HTML5 Drag & Drop API + Touch Events
- CSS Flexbox / Media Queries
- localStorage Web Storage API

## 如何运行

1. 克隆或下载本项目
2. 在文件管理器中双击 `index.html`
3. 或用任意浏览器打开 `index.html` 即可

## 数据结构

所有任务以 JSON 数组存储在 `localStorage` 中，key 为 `kanban-tasks`：

```json
[
  { "id": 1716297600000, "text": "示例任务", "column": "todo" }
]
```

`column` 可选值：`"todo"` | `"in-progress"` | `"done"`
